OilPriceAPI Excel Starter Workbook

Paste your API key into one cell and start with WTI and Brent rows

The starter workbook is the simple first-use path for Windows desktop Excel. It does not require manifest XML, macros, Developer tools, or add-in sideloading.

Download Starter Workbook Get API Key View on GitHub

First Use

  • Open the workbook in Windows desktop Excel.
  • Paste your OilPriceAPI key into Settings!B2.
  • Go to Latest Prices.
  • If values do not populate immediately, use Formulas > Calculate Now or Data > Refresh All.

Formula refresh uses Excel WEBSERVICE and FILTERXML, which are intended for Windows desktop Excel. This starter workbook is not the recommended path for Excel for Mac or Excel for the web.

What It Includes

  • WTI and Brent latest-price starter rows.
  • Visible worksheet messages for missing key, auth, quota/rate-limit, and no-data states.
  • No embedded API key or customer data.
  • No VBA, macros, or Office add-in dependency for first use.

Developer Add-in

  1. The Office add-in manifest remains available in the GitHub repo for development and testing.
  2. Use the starter workbook first unless you specifically need to test the add-in path.
