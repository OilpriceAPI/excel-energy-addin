Excel Energy Price Comparison Add-in

Compare energy commodity prices across units and currencies

This Excel add-in allows analysts to fetch real-time energy commodity prices and convert them to equivalent energy units for direct comparison.

Get API Key (Free) View on GitHub Documentation

Features

  • ✅ Real-time price fetching from OilPriceAPI
  • ✅ Energy unit conversions ($/MMBtu)
  • ✅ Support for Brent, WTI, Natural Gas, Coal, and more
  • ✅ Multi-currency support (USD, GBP, EUR)
  • ✅ Data export for analysis

Installation

  1. Download the manifest file from GitHub
  2. Open Microsoft Excel (2016 or later)
  3. Go to Insert → Get Add-ins → Upload My Add-in
  4. Select the manifest.xml file
  5. Get your free API key from OilPriceAPI.com
