OilPriceAPI Excel API Runner

One visible sheet. Paste an API key, edit the endpoint path or query string, and return a normalized results table from Excel.

The formula workbook is available for Windows desktop Excel. Mac and web users should use the browser export on oilpriceapi.com/excel until the Power Query or add-in path is verified.

First Use

  • Open the workbook in Windows desktop Excel.
  • If Excel asks, choose Enable Editing and Enable Content. Microsoft controls those prompts for downloaded workbooks that call web endpoints.
  • Paste your OilPriceAPI key into the highlighted key cell.
  • Use the default WTI lookup, or edit the visible endpoint path and query string cells.
  • Press Enter.
  • If values do not populate immediately, use Formulas > Calculate Now.

Formula refresh uses Excel WEBSERVICE and FILTERXML, which are intended for Windows desktop Excel. This starter workbook is not the recommended path for Excel for Mac or Excel for the web.

What It Includes

  • One-page API runner with visible API key, endpoint path, and query string cells.
  • Starter examples for latest price, price history, and futures curve endpoint shapes.
  • Visible worksheet messages for missing key, auth, quota/rate-limit, and no-data states.
  • No embedded API key or customer data.
  • No VBA, macros, or Office add-in dependency for first use.

Developer Add-in
