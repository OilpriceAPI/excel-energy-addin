Privacy Policy

OilPrice Excel Add-in

Effective Date: October 7, 2025

Last Updated: May 12, 2026

Summary: This add-in stores your API key in the Office add-in runtime where available and uses it to fetch OilPriceAPI data. We do not collect spreadsheet contents or workbook files.

1. Information We Collect

1.1 Information You Provide

  • API Key: When you configure the add-in, you provide an OilPriceAPI API key. The add-in stores this key in Office runtime storage where available, with local browser storage used only as a development fallback.
  • Formula Inputs: Formula arguments such as commodity codes or endpoint paths are sent to OilPriceAPI only when Excel recalculates a formula.

1.2 Automatically Collected Information

  • API Usage Logs: When you use the add-in to fetch prices, OilPriceAPI logs your API requests (IP address, timestamp, endpoint accessed) as part of our standard API service. This is governed by the OilPriceAPI Privacy Policy.
  • No Analytics: This add-in does not use Google Analytics, Microsoft Application Insights, or any other tracking service.

1.3 Information We Do NOT Collect

  • Personal identification information (name, email, phone)
  • Spreadsheet data or formulas
  • File names or workbook contents
  • Device information beyond standard HTTP headers
  • Location data

2. How We Use Your Information

Data Type Purpose Storage Location
API Key Authenticate requests to OilPriceAPI Office runtime storage where available; local browser storage fallback in development
Formula Inputs Request the selected OilPriceAPI endpoint Worksheet cells controlled by the user
API Request Logs Monitor usage, prevent abuse, improve service OilPriceAPI servers (see OilPriceAPI Privacy Policy)

3. Data Storage and Security

3.1 Add-in Storage

Your API key is stored by the add-in runtime where available. In development fallback mode, browser local storage may be used. This data:

  • Remains on your device only
  • Is not transmitted to our servers
  • Persists until you clear it from the add-in or remove the add-in storage
  • Can be deleted at any time from the add-in key manager

3.2 Data Transmission

When you fetch commodity prices, the add-in makes HTTPS requests to:

  • https://api.oilpriceapi.com/v1/prices/latest

These requests include your API key in the Authorization header. All communication uses industry-standard TLS 1.2+ encryption.

3.3 Third-Party Services

This add-in integrates with:

  • OilPriceAPI - Commodity price data provider. See OilPriceAPI Privacy Policy.
  • GitHub Pages - Static hosting for add-in assets. See GitHub Privacy Statement.

4. Data Retention

  • Add-in Data: Your API key is retained until you clear it from the add-in key manager or remove the add-in storage.
  • API Logs: OilPriceAPI retains request logs for 90 days for security and billing purposes.

5. Your Rights and Choices

5.1 Access and Deletion

You can:

  • View your API key: Open the add-in Settings panel
  • Delete your API key: Clear it in the add-in key manager
  • Request API log deletion: Contact privacy@oilpriceapi.com

5.2 Opt-Out

To stop data collection, simply uninstall the add-in from Excel.

6. Children's Privacy

This add-in is not intended for children under 13 years of age. We do not knowingly collect personal information from children.

7. International Data Transfers

OilPriceAPI servers are located in the United States. By using this add-in, you consent to the transfer of your API request data to the United States.

8. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated "Last Updated" date. Continued use of the add-in after changes constitutes acceptance of the updated policy.

9. Compliance

  • California Consumer Privacy Act (CCPA)
  • General Data Protection Regulation (GDPR) - EU users
